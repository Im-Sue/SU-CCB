# 贡献者公约行为准则

## 我们的承诺

我们作为成员、贡献者和维护者，承诺让参与我们社区的每个人都能获得无骚扰的体验，
不论年龄、体型、可见或不可见的残障、族裔特征、性别特征、性别认同和表达、
经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、种姓、肤色、宗教，
或性取向。

我们承诺以有助于开放、友善、多元、包容和健康社区的方式行动和互动。

## 我们的标准

有助于营造积极环境的行为示例包括：

- 对他人表现出同理心和善意。
- 尊重不同的意见、观点和经验。
- 给予并优雅地接受建设性反馈。
- 为错误承担责任，向受影响的人道歉，并从经验中学习。
- 关注整体社区的最佳利益，而不只是个人利益。

不可接受的行为示例包括：

- 使用性化语言或图像，以及任何形式的性关注或性挑逗。
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击。
- 公开或私下骚扰。
- 未经明确许可发布他人的私人信息，如住址或邮箱地址。
- 其他在专业环境中可合理认为不合适的行为。

## 执行责任

社区维护者负责澄清和执行可接受行为标准，并会针对他们认为不适当、威胁性、
冒犯性或有害的行为采取适当且公平的纠正措施。

社区维护者有权删除、编辑或拒绝与本行为准则不一致的评论、commit、代码、wiki 编辑、
issue 和其他贡献，并会在适当时说明管理决定的理由。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间代表社区时。代表社区的例子
包括使用官方邮箱、通过官方社交媒体账号发言，或在线上、线下活动中作为指定代表。

## 执行

可通过 GitHub issue 或维护者联系渠道举报滥用、骚扰或其他不可接受行为。

联系方法：请在 GitHub issue 中标注 `conduct`，或联系项目维护者 @SU-CCB-maintainers。

所有投诉都会被及时、公平地审查和调查。

所有社区维护者都有义务尊重举报人的隐私和安全。

## 执行指南

社区维护者会按照以下社区影响指南决定他们认为违反本行为准则行为的后果。

### 1. 纠正

社区影响：使用不合适的语言或其他在社区中被认为不专业或不受欢迎的行为。

后果：由社区维护者发出私人书面警告，说明违规性质，并解释为什么该行为不合适。
可能要求公开道歉。

### 2. 警告

社区影响：单次事件或一系列行为导致违规。

后果：警告并说明继续违规的后果。指定期限内不得与相关人员互动，包括主动联系执行者。
违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

社区影响：严重违反社区标准，包括持续不当行为。

后果：在指定期限内临时禁止与社区进行任何互动或公开交流。期间不得与相关人员进行公开
或私下互动，包括主动联系执行者。违反这些条款可能导致永久封禁。

### 4. 永久封禁

社区影响：表现出违反社区标准的模式，包括持续不当行为、骚扰个人，或攻击或贬损某类人群。

后果：永久禁止在社区内进行任何公开互动。
