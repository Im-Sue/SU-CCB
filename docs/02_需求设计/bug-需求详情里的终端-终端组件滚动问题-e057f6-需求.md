## 十三、风险

- **client-local 历史**:快照模型下不同客户端/打开时刻看到的「历史」不同;若用户预期「同一事实历史」,只有 B(后端 ring buffer)能收敛。
- **alt-screen TUI 语义**:全屏重绘 app 的「历史」= 快照序列,B/E 的「干净行历史」对 TUI 输出价值受限,须预期管理(演示后再定档)。
- **性能**:A 档 scrollback 膨胀(变化帧每帧推一屏);B 档动协议 + 后端常驻状态 + 重连补发;150ms 帧率是放大器。
- **回归面**:共享 surface 连带 main 终端弹窗;`06c62eb` 双底部贴底/仲裁逻辑;粘贴/复制链路。
- **可访问性**:隐藏原生滚条后 PageUp/触控/键盘历史入口可能缺失(随 P2 真浏览器复现一并查)。
