# 项目管理模块文档

## 1. 模块目标

| 项 | 内容 |
|---|---|
| 模块名 | 项目管理 |
| 目标 | 管理本地项目的接入、基础信息、初始化状态与重扫入口 |
| V1 价值 | 所有能力的入口模块 |

## 2. 功能范围

| 功能 | V1 状态 |
|---|---|
| 创建项目 | 必做 |
| 编辑项目基础信息 | 必做 |
| 校验本地路径 | 必做 |
| 展示初始化状态 | 必做 |
| 手动初始化入口 | 必做 |
| 手动重扫入口 | 必做 |
| 删除项目 | 可后置 |

## 3. 页面与组件

| 页面/组件 | 作用 |
|---|---|
| 项目列表页 | 展示项目卡片 |
| 项目创建弹窗 | 输入名称、路径、简介 |
| 项目概览头卡 | 展示路径与状态 |
| `ProjectCard` | 项目摘要组件 |
| `InitStatusBadge` | 初始化状态标签 |

## 4. 输入输出

| 类型 | 内容 |
|---|---|
| 输入 | 项目名、本地路径、简介 |
| 输出 | `Project` 记录、初始化状态、最近扫描时间 |

## 5. 依赖

| 依赖模块 | 原因 |
|---|---|
| 同步模块 | 初始化与重扫会生成运行记录 |
| 文档模块 | 项目扫描后进入文档索引 |
