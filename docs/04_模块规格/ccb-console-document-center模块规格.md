# 文档中心模块文档

## 1. 模块目标

| 项 | 内容 |
|---|---|
| 模块名 | 文档中心 |
| 目标 | 分类浏览、搜索、阅读 `spec / plan / task` 文档 |
| V1 价值 | 建立“文档是真源”的可视化能力 |

## 2. 功能范围

| 功能 | V1 状态 |
|---|---|
| 文档扫描结果展示 | 必做 |
| 类型筛选 | 必做 |
| 目录树浏览 | 必做 |
| 在线阅读 | 必做 |
| 搜索 | 必做 |
| 复杂正文编辑 | 不做 |

## 3. 页面与组件

| 页面/组件 | 作用 |
|---|---|
| 文档中心页 | 主页面 |
| `DocumentTree` | 左侧目录树 |
| `DocumentList` | 中间文档列表 |
| `DocumentReader` | 右侧正文阅读区 |
| `ParseStatusBadge` | 展示解析状态 |

## 4. 输入输出

| 类型 | 内容 |
|---|---|
| 输入 | 项目路径下的文档文件 |
| 输出 | `Document` 索引、正文预览、关联任务跳转 |

## 5. 依赖

| 依赖模块 | 原因 |
|---|---|
| 同步模块 | 文档索引来源于扫描与解析 |
| 任务模块 | 文档需能跳转到关联任务 |
