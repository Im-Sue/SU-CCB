# 同步与索引模块文档

## 1. 模块目标

| 项 | 内容 |
|---|---|
| 模块名 | 同步与索引 |
| 目标 | 负责初始化记录、扫描、解析、归并与异常可观测性 |
| V1 价值 | 保证文档视图和任务视图可维护、可解释 |

## 2. 功能范围

| 功能 | V1 状态 |
|---|---|
| 初始化记录 | 必做 |
| 扫描记录 | 必做 |
| 解析记录 | 必做 |
| 归并记录 | 必做 |
| 异常摘要展示 | 必做 |
| 重试入口 | 可做简版 |
| 完整文件监听 | 可后置 |

## 3. 页面与组件

| 页面/组件 | 作用 |
|---|---|
| 运行记录页 | 主页面 |
| `RunLogTable` | 运行记录表格 |
| `SyncJobList` | 同步作业列表 |
| `ErrorStateCard` | 异常提示 |

## 4. 输入输出

| 类型 | 内容 |
|---|---|
| 输入 | 初始化动作、扫描动作、解析动作、归并动作 |
| 输出 | `SyncJob` 记录、错误摘要、项目最新同步状态 |

## 5. 依赖

| 依赖模块 | 原因 |
|---|---|
| 项目模块 | 同步作业按项目组织 |
| 文档模块 | 扫描与解析文档 |
| 任务模块 | 归并任务视图 |
