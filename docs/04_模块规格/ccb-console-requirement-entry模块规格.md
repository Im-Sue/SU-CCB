# 需求入口模块文档

## 1. 模块目标

| 项 | 内容 |
|---|---|
| 模块名 | 需求入口 |
| 目标 | 为项目提供新增需求与文档骨架生成入口 |
| V1 价值 | 建立新工作的统一创建路径 |

## 2. 功能范围

| 功能 | V1 状态 |
|---|---|
| 新增需求 | 必做 |
| 选择输出策略 | 必做 |
| 生成骨架预览 | 必做 |
| 生成 spec 骨架 | 必做 |
| 生成 spec + plan + task 骨架 | 必做 |
| 直接调用 CCB 命令生成 | 不做，后续再接 |

## 3. 页面与组件

| 页面/组件 | 作用 |
|---|---|
| 需求入口页 | 主页面 |
| `RequirementForm` | 表单 |
| 输出策略卡片 | 选择骨架生成方式 |
| 结果预览区 | 展示将创建的文档与任务 |

## 4. 输入输出

| 类型 | 内容 |
|---|---|
| 输入 | 标题、背景、目标、补充说明、输出策略 |
| 输出 | `Requirement` 记录、基础文档骨架、初始任务 |

## 5. 依赖

| 依赖模块 | 原因 |
|---|---|
| 项目模块 | 需求必须归属项目 |
| 文档模块 | 生成文档骨架 |
| 任务模块 | 创建初始任务对象 |
