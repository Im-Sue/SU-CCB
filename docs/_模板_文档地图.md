## 为什么有这张表

> 统一回答:哪些文档现在还作数、哪些只是历史参考。是判断"哪篇是孤儿"的依据。

---

## 三种状态

| 状态 | 含义 |
|------|------|
| 生效中 | 现在还作数,可作为当前依据 |
| 历史 | 被取代了,留着参考,不能当现在的依据 |
| 归档 | 只是历史快照,不参与现在的讨论 |

## 状态怎么变(改实体或位置,不改这张表、也不改文档 status 字段)

1. **实体推进** → 改需求/任务/ADR 的状态(命令维护)→ 它名下文档的显示跟着变
2. **文档退役** → 移进 `99_归档/`(位置)→ 显示"归档"
3. **常青文档** → 一直"生效中",更新内容即可,无需状态字段
